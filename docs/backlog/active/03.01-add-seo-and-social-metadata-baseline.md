# Story 03.01: Add SEO and Social Metadata Baseline

## Objective

Establish foundational SEO and social-sharing metadata across core website routes.

## Scope

1. Add canonical metadata defaults in shared layout.
2. Add Open Graph and Twitter card metadata for Home, Blog index/post, and Docs index/doc routes.
3. Ensure page-level title/description metadata remains route-specific and stable.
4. Document metadata conventions for future content additions.

## Acceptance Criteria

- Core routes emit consistent baseline SEO/social tags.
- Metadata renders correctly in static output.
- Existing URLs and content structure remain unchanged.
